# Gaza War Predictions

This notebook contains the script used in the <a href="https://app.powerbi.com/reportEmbed?reportId=6e75fe72-d56c-47e3-9b89-e39d910eadce&autoAuth=true&ctid=9db3f0d7-245e-420f-8d3e-81ebe1ae6ff9">Gaza War Predictions dashboard</a>.

Predictions were calculated using the ARIMA method (autoregressive integrated moving average), applied to Gaza War data from <a href="https://techforpalestine.org/">Tech For Palestine</a> to predict the next 3 to 6 months' death, injury, building damage, and destroyed building counts in the Gaza Strip.
